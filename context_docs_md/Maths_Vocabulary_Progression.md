# Maths Vocabulary Progression

> **Note:** This document is entirely image-based in the original PDF. It contains visual vocabulary progression tables showing the mathematical vocabulary to be taught and used at each year group from EYFS through to Year 6.
>
> The document maps key vocabulary across all strands of the mathematics curriculum (number, calculation, fractions, measurement, geometry, statistics), showing how vocabulary builds progressively across year groups to support pupils' mathematical communication and reasoning.

## Purpose

Providing children with the correct mathematical vocabulary is a key part of the mastery approach. Maths displays in classrooms contain key vocabulary linked to current topics. This progression document ensures:

- Consistent vocabulary use across the school
- Clear expectations for vocabulary at each year group
- Support for developing mathematical reasoning through precise language
- Progression from informal to formal mathematical language
